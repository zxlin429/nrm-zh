# nrm-zh

中文版npm镜像源管理工具(中文版nrm),全中文提示,减轻中国程序员心智负担

## 安装

### npm
```
npm install npm-zh
```

### yarn
```
yarn add npm-zh
```

### pnpm
```
pnpm install npm-zh
```

## 使用

nrm-zh -h 或者 help => 查看命令

nrm-zh -V => 查看版本号

nrm-zh ls => 查看镜像源列表

nrm-zh add <镜像源名称:必填项> <镜像源地址:必填项> => 添加镜像源

nrm-zh del <镜像源名称:必填项> => 删除镜像源

nrm-zh use <镜像源名称:必填项> => 切换镜像源

nrm-zh add <镜像源名称:必填项> <镜像源地址:必填项> => 添加镜像源
